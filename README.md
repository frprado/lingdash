# Linguistics Research — A Bibliometric Study

Which languages get studied in peer-reviewed linguistics is not a neutral question. It reflects patterns of academic prestige, institutional funding, and the afterlives of colonial infrastructure. This project assembles a corpus of linguistics journal articles indexed in [OpenAlex](https://openalex.org) and analyses where the research comes from, which languages it focuses on, and how the breadth of the field's agenda has changed over six decades.

**[Interactive dashboard](https://huggingface.co/spaces/ycalx/lingdash)** — for methodology and findings see [ABOUT.md](ABOUT.md).
